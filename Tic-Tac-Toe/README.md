# hs-Tic-Tac-Toe
Noughts and Crosses
<hr>

Input number from 1 - 3

Grid structure: <br>
(1, 3) (2, 3) (3, 3) <br>
(1, 2) (2, 2) (3, 2) <br>
(1, 1) (2, 1) (3, 1) <br>
